# 删除随机壁纸 + 快捷方式图标自定义（实施计划）

> 依据：`doc/2026-03-26-删除随机壁纸-快捷方式自定义图标与显示名-设计.md`

## Goal

- 删除随机壁纸（Lorem Picsum）功能：UI/逻辑/隐私政策/历史缓存。
- 快捷方式支持：
  - 显示名称（沿用现有能力，统一文案）
  - 图标模式：**自动** / **内置** / **上传**
  - 上传图标使用 IndexedDB 持久化，避免 localStorage 膨胀
  - 拖拽排序/删除/重置时保持 icon 配置一致，并清理上传 blob

## Files（预计修改）

### 删除随机壁纸

- Modify: `index.html`（删除 `#randomImageTrigger` 按钮）
- Modify: `scripts/wallpaper.js`（移除随机壁纸逻辑 + 启动时清理 random 类型缓存）
- Modify: `privacy-policy.html`（移除 Random Wallpapers API 条目）

### 快捷方式图标自定义

- Modify: `index.html`（快捷方式编辑页：每条 entry 增加 icon 控件的 DOM 容器）
- Modify: `style.css`（icon 控件布局样式）
- Modify: `scripts/shortcuts.js`
  - 新增：icon 的 localStorage 键读写
  - 新增：`ShortcutIconDB`（IndexedDB store: `shortcutIcons`）
  - 改造：render 图标优先级（upload → builtin → auto）
  - 改造：排序保存/删除/重置，纳入 icon 字段并清理 upload blob
- Modify: `locales/*`（如需新增文案 key：例如“显示名称/图标/自动/内置/上传/选择图标/上传图标”）

## Data keys（新增）

localStorage：

- `shortcutIconType{i}`: `auto | builtin | upload`
- `shortcutIconValue{i}`:
  - builtin：内置 key
  - upload：`shortcutIconUpload:{uuid}`

IndexedDB：

- DB: `ShortcutIconDB`（version 1）
- Store: `shortcutIcons`
- Key: `shortcutIconUpload:{uuid}`
- Value: Blob（建议缩放到 128×128 或 192×192）
